HTML 5 Features
  • Semantic Elements
    • When constructing an HTML script, the term “semantic” refers to the meaning of the text.
  • Audio and Video Support
    • One of the many HTML 5 features is the support for audio and video. It has reduced the hassle of relying upon third-party services such as Adobe Flash player.
  • Local Storage
    • It is a modern feature of HTML and several browsers that typically store data in the users browsers and can access them with the help of JavaScript APIs. This feature is useful for creating offline applications where data is need to be stored locally.
  • Web Sockets
    • In the previous versions of HTML, when a client sends a request to the backend server, the server then responds afterward. However, in HTML 5 we can establish a bidirectional live communication between the server and the client (a web browser) to reduce the latency in the responses.
